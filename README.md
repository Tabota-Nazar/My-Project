# Telegram Movie Bot

Це Telegram-бот для роботи з фільмами.  
Бот дозволяє:
- переглядати список фільмів
- додавати нові фільми
- редагувати опис фільму
- видаляти фільми
- оцінювати фільми
- шукати фільми за назвою
- фільтрувати фільми за жанром або роком
- отримувати рекомендації кращих фільмів

---

## Команди бота

| Команда | Опис |
|---------|------|
| `/start` | Почати розмову з ботом |
| `/films` | Перегляд списку фільмів |
| `/add_movie` | Додати новий фільм |
| `/search_movie` | Пошук фільму за назвою |
| `/filter_movies` | Фільтр фільмів за жанром або роком |
| `/edit_movie` | Редагування опису фільму |
| `/delete_movie` | Видалення фільму |
| `/rate_movie` | Оцінити фільм |
| `/recommend_movie` | Отримати рекомендацію кращого фільму |

---

## Як запустити

1. Клонувати або завантажити проєкт на свій комп'ютер.
2. Створити віртуальне середовище (не обов'язково, але бажано):


python -m venv .venv

Активувати віртуальне середовище:
.venv\Scripts\Activate.ps1

Встановити бібліотеки:
pip install -r requirements.txt

Додати токен вашого бота у файл config.py:
BOT_TOKEN = "тут_ваш_токен"

Запустити бота:
python bot.py

Запуск тестів:
python -m pytest -v test_data.py
